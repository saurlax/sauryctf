# 管理端快速建赛

这份说明面向第一次把平台真正跑起来的管理员。

## 推荐顺序

1. 使用 `/login` 登录管理账号。
2. 进入 `/console/admin`。
3. 先用“创建基础比赛”或“创建动态实例比赛”生成比赛骨架。
4. 检查比赛时间、报名模式、公开状态和挂题情况。
5. 用普通账号完成注册、组队、报名和一次题目提交。
6. 回到管理端确认报名状态、实例状态和排行榜更新是否正常。

## 快速入口

- `创建基础比赛`
  - 自动创建一场公开比赛和一道基础题。
  - 适合优先打通报名、提交和排行榜闭环。
- `创建动态实例比赛`
  - 自动创建一场公开比赛和一道队伍独立入口动态题。
  - 适合检查实例租约、入口回填和续期链路。
- `创建容器实例比赛`
  - 自动创建一场公开比赛和一道容器实例题。
  - 适合在本机已经具备 Docker 运行条件时检查真实实例入口。

## 使用建议

- 模板创建完成后，应及时替换题面、Flag、公告和规则补充。
- 如果当前只是做首轮联通验证，优先关注：
  - 登录是否正常
  - 报名是否可达
  - Flag 提交是否记分
  - 排行榜是否更新
  - 动态题入口是否能正确回填
