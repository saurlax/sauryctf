# 用户管理

`/console/admin/users` 提供最小可用的账号管理入口，仅对 `admin` 和 `super_admin` 开放。

当前能力：

- 查看全部用户
- 查看用户角色、状态、邮箱和创建时间
- 按用户单独更新角色
- 按用户单独更新状态

当前约束：

- 当前登录账号不能把自己改成封禁
- 只有 `super_admin` 可以管理 `super_admin`
- 非 `super_admin` 不能把其他用户提升为 `super_admin`
- 前端会提前隐藏这类无效选项：
  - 当前账号自己的状态下拉里不会出现 `封禁`
  - 非 `super_admin` 的角色下拉里不会出现 `super_admin`

建议用法：

- 只在确有运维或权限调整需求时修改角色
- 封禁前先确认该账号是否仍承担比赛管理职责
- 默认管理员完成初始化后，应优先在“账号安全”页更新密码
