# 队伍管理

当前 `/console/team` 已支持最小可用的队伍管理流程。

## 当前能力

- 创建队伍
- 通过邀请码加入队伍
- 复制邀请码分享给队友
- 普通队员主动退出队伍
- 队长移除其他成员
- 显示当前锁定中的比赛，解释为什么当前不能调整队伍

## 当前约束

- 创建者自动成为队长
- 队长当前不能直接退出队伍
- 如果队伍已经通过任意一场仍未结束的比赛报名，队伍会被临时锁定：
  - 不能再通过邀请码加入新成员
  - 普通成员不能退队
  - 队长不能移除成员
- 移除成员使用现有接口 `DELETE /api/teams/{teamId}/members/{memberId}`

## 当前前端原则

- 保持 Nuxt UI 极简布局
- 队长和队员看到的操作保持明确分层
- 优先服务“创建队伍 -> 邀请队友 -> 报名比赛”的主流程
- 如果队伍已经被比赛锁定，优先直接展示锁定原因和关联比赛，而不是让用户靠接口报错理解规则
