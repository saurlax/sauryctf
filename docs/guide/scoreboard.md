# 排行榜

当前排行榜已具备两层信息：

- 队伍总榜
  - 排名
  - 队伍名
  - 总分
  - 解题数
  - 最后一次解题时间
- 分题统计
  - 题目名称
  - 分类
  - 当前分值
  - 解出队伍数
  - 一血队伍
  - 按分类分组展示
- 封榜信息
  - 当前是否已封榜
  - 封榜时间
- 分组信息
  - 当前筛选分组
  - 比赛可用分组列表

## 当前实现

- 接口：`GET /api/games/{id}/scoreboard`
- 返回：
  - `entries`：队伍总榜
  - `challenges`：分题统计
  - `is_frozen`：公开榜单是否已封榜
  - `freeze_time`：封榜时间
  - `division`：当前筛选的分组
  - `divisions`：比赛配置的分组列表
- 前端展示：
  - 队伍总榜保留表格
  - 如果比赛启用了分组，可切换“全部分组”或某个具体分组
  - 如果比赛已进入封榜时间，会展示封榜提示
  - 分题统计按 `category` 分组后再展示卡片

## 与 GZCTF 的对齐方向

这一版先实现最小可用的比赛榜单视图，并补上基础封榜能力，后续可以继续补：

- 更完整的 blood 信息
- 更细的时间线展示
- ETag / 缓存优化
