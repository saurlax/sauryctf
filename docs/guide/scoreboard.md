# 排行榜

当前排行榜已具备两层信息：

- 队伍总榜
  - 排名
  - 队伍名
  - 总分
  - 解题数
  - 最后一次解题时间
- 分题统计
  - 题目名称
  - 分类
  - 当前分值
  - 解出队伍数
  - 一血队伍

## 当前实现

- 接口：`GET /api/games/{id}/scoreboard`
- 返回：
  - `entries`：队伍总榜
  - `challenges`：分题统计

## 与 GZCTF 的对齐方向

这一版先实现最小可用的比赛榜单视图，后续可以继续补：

- 按分类分组展示题目统计
- 更完整的 blood 信息
- 时间线或封榜机制
- ETag / 缓存优化
