# 控制台首页

当前 `/console` 已从占位页调整为可用的导航首页。

## 当前能力

- 展示我的队伍状态
- 展示公开比赛统计
- 展示“我的比赛”入口，按报名状态聚合当前用户最需要处理的比赛
- 展示最近比赛列表
- 提供最近比赛快捷入口
- 管理员可直接跳转到 `/console/admin`
- 普通用户如果误入 `/console/admin`，会被前端中间件直接带回 `/console`

## 当前设计

- 只复用现有接口：
  - `GET /api/teams/my`
  - `GET /api/games`
  - `GET /api/games/{id}/participation`
- 不新增后端聚合接口
- 前端保持 Nuxt UI 极简卡片式布局
- “我的比赛”当前会优先突出：
  - 待补 Writeup
  - 待审核 / 已拒绝
  - 已报名且正在进行中的比赛
  - 尚未报名但仍可进入的公开比赛

## 后续可继续补充

- 展示最近解题记录
- 管理员比赛状态快捷切换

## 当前公开可见性

- 公开比赛列表只展示：
  - `is_public = true`
  - 且不处于 `draft` 的比赛
- 草稿比赛仍然保留在管理端中，方便管理员继续配置
