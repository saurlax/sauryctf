# 管理端

当前管理端先实现了一个最小闭环页面：

- 路径：`/console/admin`
- 仅 `admin` / `super_admin` 角色可见
- 能力：
  - 创建比赛
  - 修改比赛名称、描述、时间
  - 修改比赛状态与公开性
  - 创建题目
  - 将已有题目挂载到比赛
  - 查看某场比赛当前已挂载题目
  - 将题目从比赛中移除

## 当前设计原则

- 先复用现有后端 admin API
- 前端保持极简，只做表单输入和结果反馈
- 同步加载比赛、题目列表，方便管理时直接选择
- 选中比赛后直接查看当前挂题，减少管理切换成本
- 比赛基础信息与比赛状态拆成两个独立表单，避免误操作
- 比赛状态编辑先只开放 `draft / active / ended` 与 `is_public`
- 不在这一版里加入复杂的编辑、删除、批量操作
