# 比赛列表

当前 `/games` 已作为公开比赛的主入口页。

## 当前能力

- 展示所有公开且已开放展示的比赛
- 顶部展示比赛总数、进行中数量、已结束数量、我的已报名数量
- 支持按关键字搜索：
  - 比赛名称
  - 比赛描述
  - 比赛公告
  - 比赛分组
- 支持按状态筛选：
  - 全部状态
  - 进行中
  - 已结束
- 每张比赛卡片会展示：
  - 比赛状态
  - 报名方式、人数限制、分组、赛后策略与 Writeup 要求
  - 当前用户自己的参赛状态
  - 对应操作入口

## 当前设计

- 比赛列表只读取公开比赛接口，不混入管理端视图
- 参赛状态继续复用统一的 `resolveParticipationMeta`
- 列表页优先承担“发现比赛、筛选比赛、查看当前状态”的入口职责
- 比赛卡片当前优先使用结构化摘要行：
  - 用简短标签展示报名方式、人数限制、分组和赛后策略
  - 避免把规则说明堆成多段连续正文
- 空结果时会使用正式空态提示当前没有匹配比赛，并直接提供清空筛选和对应入口

## 当前约束

- `draft` 比赛不会出现在公开列表
- 未登录用户也能浏览比赛基础信息
- 只有登录后才会显示与当前账号相关的真实报名状态
