# 比赛列表

当前 `/games` 已作为公开比赛的主入口页。

## 当前能力

- 展示所有公开且已开放展示的比赛
- 顶部展示比赛总数、进行中数量、已结束数量、我的已报名数量
- 支持按关键字搜索：
  - 比赛名称
  - 比赛描述
  - 比赛公告
  - 比赛分组
- 支持按状态筛选：
  - 全部状态
  - 进行中
  - 已结束
- 每张比赛卡片会展示：
  - 比赛状态
  - 报名方式
  - 队伍人数上限
  - 比赛分组
  - 是否支持赛后练习
  - 是否要求 Writeup
  - Writeup 截止时间
  - 当前用户自己的参赛状态
  - 进入下一步的推荐按钮

## 当前设计

- 比赛列表只读取公开比赛接口，不混入管理端视图
- 参赛状态继续复用统一的 `resolveParticipationMeta`
- 列表页优先承担“发现比赛、筛选比赛、判断下一步”的入口职责
- 空结果时会直接提示当前筛选条件下没有匹配比赛

## 当前约束

- `draft` 比赛不会出现在公开列表
- 未登录用户也能浏览比赛基础信息
- 只有登录后才会显示与当前账号相关的真实报名状态
